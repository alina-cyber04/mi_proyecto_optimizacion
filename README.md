# mi_proyecto_optimizacion

Proyecto de ejemplo para experimentos de optimización.

Estructura:

- `src/optimizacion/` : módulos de optimización (línea de búsqueda, BFGS, gradiente, utilidades JSON y graficado).
- `notebooks/experimentos.ipynb` : notebook principal con las experimentos y visualizaciones.
- `data/resultados/` : carpeta para guardar JSON de resultados y figuras.

Cómo usar:

1. Instala dependencias: `pip install -r requirements.txt`.
2. Abre `notebooks/experimentos.ipynb` en Jupyter.
3. Ejecuta las celdas para reproducir los experimentos.
